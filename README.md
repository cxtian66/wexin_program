# 微信记账小程序

一个简单易用的微信记账小程序，帮助用户记录日常收支，掌握个人财务状况。

## 项目概述

本项目是一个基于微信小程序原生开发的记账应用，使用微信云开发作为后端服务，无需搭建服务器。通过本小程序，用户可以方便地记录日常收支、查看记账历史、分析消费数据，从而更好地管理个人财务。

## 功能特性

- **快速记账**：支持快速添加收支记录，选择分类、金额、日期等信息
- **分类管理**：内置多种收支分类，支持自定义分类
- **记账历史**：查看所有记账记录，支持按日期、分类筛选
- **统计分析**：提供收支概览、分类统计、月度趋势等数据可视化
- **数据管理**：支持数据导出、导入、清空等操作
- **用户体验**：响应式设计，适配不同设备尺寸，操作简单直观

## 技术栈

- **前端**：微信小程序原生开发（WXML、WXSS、JavaScript）
- **后端**：微信云开发（云函数、云数据库、云存储）
- **数据可视化**：Canvas图表绘制
- **开发工具**：微信开发者工具

## 快速开始

### 环境要求

- 微信开发者工具
- 微信小程序账号
- 微信云开发环境

### 项目初始化

1. 克隆本项目到本地
2. 在微信开发者工具中导入项目
3. 配置云开发环境：
   - 在 `app.js` 中修改云开发环境ID
   - 初始化云开发环境
4. 创建数据库集合：
   - `users`：存储用户信息
   - `records`：存储记账记录
   - `categories`：存储分类信息
5. 导入默认分类数据到 `categories` 集合

### 开发命令

- **预览**：在微信开发者工具中点击预览按钮
- **调试**：在微信开发者工具中开启调试模式
- **上传**：在微信开发者工具中点击上传按钮

## 云开发配置

### 数据库集合

1. **users**：存储用户信息
   - openid：用户微信openid
   - nickName：用户昵称
   - avatarUrl：用户头像
   - createdAt：创建时间
   - updatedAt：更新时间
2. **records**：存储记账记录
   - openid：用户微信openid
   - type：类型（1-收入，2-支出）
   - amount：金额
   - category：分类ID
   - categoryName：分类名称
   - date：记账日期
   - time：记账时间
   - remark：备注
   - createdAt：创建时间
   - updatedAt：更新时间
3. **categories**：存储分类信息
   - name：分类名称
   - type：类型（1-收入，2-支出）
   - icon：分类图标
   - color：分类颜色
   - sort：排序顺序
   - isDefault：是否默认分类

### 云函数

- **login**：获取用户openid
